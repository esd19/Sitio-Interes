# Sitio básico: Temas de Interés

Este proyecto cumple con los requisitos solicitados:
- Todo el sitio dentro de un **contenedor** (`.app-container`).
- **Header** con logo a la izquierda.
- En el **body**, enlace a un video de un servicio de streaming (YouTube).
- **Footer** con datos de identificación (reemplazar placeholders).
- **Imagen de fondo** (`assets/bg.png`).

## Uso local
Abre `index.html` en tu navegador. No requiere servidor.

## Despliegue sugerido
- **Render (Static Site)**: conecta tu repo y crea un **Static Site**. Deja vacío “Build Command” y usa la raíz como “Publish Directory” si es HTML/CSS/JS puros.
- **Railway (Static Hosting)**: usa el flujo de **Static Hosting** o el template NGINX y coloca tus archivos en la carpeta indicada.
